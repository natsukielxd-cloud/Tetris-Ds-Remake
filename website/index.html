About

Tetris DS Demake is a full-featured Tetris game inspired by the Nintendo DS classic. Built in Python with pygame, it recreates the authentic level progression, music system, and visual style of the original while adding modern features like rebindable controls, controller support, LAN multiplayer, and a complete profile system.

Created with love by Nattyelxdyt.

Features

Classic Tetris

Full SRS wall kicks, lock delay, ghost piece, hold, and next preview. 29 levels with authentic speed curve.

9 Visual Styles

Each level range has its own background, music, and color scheme — from Mario Tetris to Duck Hunt to Ancient Tetris.

Custom Music

Drop in your own BGM files, assign them by keyword to styles, or let the game generate procedural tracks.

Multiplayer

LAN discovery, direct IP, playit.gg tunneling, and headless server support. Battle friends with garbage mechanics.

Controller Support

Full Xbox controller support with rebindable buttons, D-pad, analog stick, and auto-repeat DAS.

Customization

Background images, cat photos (Flora/Natsuki), custom music folders, and profiles with stats tracking.
